O mně

🎯 Můj příběh

Jsem vystudovaný policista, který se původně šel učit řemeslu. Ale vždycky jsem měl technické vlohy od dětství a programování jsem objevil poměrně pozdě, ale o to intenzivněji. Dlouho jsem hledal směr, který by mě opravdu bavil a naplňoval. Když jsem poprvé napsal "Hello World" v Javě, věděl jsem, že jsem našel to svoje.

Začal jsem jako naprostý začátečnik - žádné informatické vzdělání, žádná předchozí zkušenost s kódem. Jen zvědavost a odhodlání naučit se něco nového. Prvních pár měsíců bylo frustrujících, ale postupně se začaly skládat puzzle pieces a programování se stalo mou vášní.

Co mě na Javě fascinuje? Je to jazyk, který má jasnou strukturu, ale zároveň obrovské možnosti. Spring Boot ecosystem, enterprise patterns, čistý kód - to vše mi dává smysl a baví mě v tom pokračovat každý den.

🚀 Kde jsem teď

Momentálně se učím a zdokonaluji v backend development s Javou 21 a Spring Boot. Mám rád čisté API, správně navržené databáze a kód, který je čitelný i za půl roku.

Věřím v pragmatický přístup - nepoužívám technologie jen proto, že jsou trendy, ale proto, že řeší konkrétní problémy. Testování, dokumentace a code review považuji za nezbytnou součást vývoje, ne za zbytečnou zátěž.

  • ✅ Píšu čitelný a udržovatelný kód
  • ✅ Testuji kritická místa aplikace
  • ✅ Dokumentuji API a architekturu
  • ✅ Používám Git s čistou historií commitů
  • ✅ Komunikuji otevřeně o problémech i řešeních

🎓 Jak se učím

Absolvoval jsem 6měsíční rekvalifikační kurz Java programátora, který mi dal pevné základy. Od té doby jsem se stal autodidaktem, což znamená, že si vše musím najít, pochopit a vyzkoušet sám. Tenhle přístup mě naučil být samostatný a důkladný. Když něco nechápu, nezůstanu u povrchu, ale jdu do hloubky, dokud to nedává smysl.

Moje oblíbené zdroje jsou oficiální dokumentace, Stack Overflow, GitHub repozitáře úspěšných projektů a občas kvalitní kurzy. Vlastní projekty jsou pro mě nejlepší učitel - když narazím na problém v reálném kódu, pamatuji si řešení mnohem lépe.

🔮 Kam směřuji

Chci se stát seniorním Java developerem, který nejen píše kvalitní kód, ale také mentoruje začátečníky a přispívá k architektonickým rozhodnutím. Zajímají mě mikroservisy, cloud technologie (AWS, Azure), AI integrace a pokročilé návrhové vzory.

Věřím, že budoucnost programování je v symbióze s umělou inteligencí - ne jako nahrazení, ale jako mocný nástroj pro rychlejší vývoj a lepší řešení problémů. Zajímá mě, jak AI integrovat do backend aplikací a využívat ji pro automatizaci.

Hledám tým, kde se pracuje chytře, otevřeně komunikuje a kde mohu růst spolu s projekty. Ideálně prostředí, kde se nebojí experimentovat, ale zároveň drží kvalitní standardy.
